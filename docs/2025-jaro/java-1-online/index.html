Java 1 (online)

Tříměsíční kurz Javy pro začátečníky

Informace o kurzu

Cílem kurzu je získání praktických zkušeností s programováním. Ovládání programátorského editoru (IDE), pochopení pojmů proměnná, metoda, objekt, třída. Algoritmizace a používání jazyka Java v praxi na malých aplikacích pro desktop (Windows, Linux, Mac).

Pro koho je kurz určen

Kurz je určen pro začátečnice, které absolvovaly Úvod do programování 1 (bez ohledu na programovací jazyk) a Úvod do programování 2 - Java v rámci Czechitas. Případně pro ty, které Javu zkoušely ve škole. Je nutné rozumět tomu, co je to proměnná, cyklus, podmínka, objekt a volání existujících metod na objektech. Není třeba umět psát vlastní třídy.

Náplň lekcí

  • Základ algoritmizace (cykly, podmínky), proměnné
  • Metody
  • Metody s parametry, proměnné
  • Metody vracející návratovou hodnotu
  • Tvorba tříd, zapouzdření a vlastnosti
  • Java Collections Framework, seznamy, množiny
  • Mapy, jednotkové testy
  • Moderní Java – Optional, lambda výrazy
  • Výjimky
  • Ukládání dat, formát JSON
  • Dědičnost, statické proměnné a metody
  • Závěr

Organizační pokyny

Kurz je každé úterý od 18:00 do 20:30 online (přes Zoom).

Materiály pro kurz Java 1, který pořádá Czechitas. Pro komunikaci používáme Discord 2655 – Java 1 – Základy 04/2025.

Je opravdu důležité, abyste mohli přijít na všechny hodiny. Druhou zásadní podmínkou jsou domácí úkoly. Budete dostávat domácí úkol na většině lekcí, proto počítej, že mimo času lekce (v úterý večer) bude třeba věnovat kurzu ještě několik hodin, například o víkendu. Domácí úkoly se ukázaly jako nejefektivnější způsob, jak se látku naučit. Máte čas s Javou pořádně experimentovat, řešit problémy a musíte si s nimi poradit. Je dokonce možné řešit úkoly ve dvojici. My vám rádi pomůžeme přes Slack, Zoom, osobně…

Úspěšná absolventka kurzu má maximálně 3 absence a musí úspěšně odevzdat všech sedm povinných domácích úkolů.

Podmínky to nejsou nijak složité, to určitě zvládnete. Těšíme se na vás 😀
